## Live: https://bartholomeas.github.io/bMusic/

## General info
Music player with features such as:
- track looping
- random playback
- volume control
- scrolling tracks
- displaying the track list

## Technologies
- HTML
- CSS
- Javascript
- Typescript
- React(Create React App)
- Tailwind CSS

	
## Additional infos
Project is created with:
* React 18.1.0
* Typescript 4.7.2
* npm 6.14.7
* Node 14.9.2
* TailwindCss 3.0.24
